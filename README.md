# လင့်ခ် သန့်ရှင်းရေး

လင့်ခ်များမှ ဒေါင့်များ `[ ]` နှင့် ကွင်းစကွင်းပိတ်များ `( )` ကို အလွယ်တကူ ဖယ်ရှားပေးသော ဝဘ်ဆိုဒ်တစ်ခု ဖြစ်သည်။

## ✨ အထူးလုပ်ဆောင်ချက်များ

- 🔗 လင့်ခ်များမှ `[` `]` ဒေါင့်များနှင့် `(` `)` ကွင်းစကွင်းပိတ်များကို အလိုအလျောက် ဖယ်ရှားပေးခြင်း
- 📋 ရှင်းလင်းပြီး လင့်ခ်ကို တစ်ချက်နှိပ်ရုံဖြင့် ကူးယူနိုင်ခြင်း
- 📊 ဝဘ်ဆိုဒ် လည်ပတ်မှု စာရင်းဇယားပြသခြင်း
- 🌐 စုစုပေါင်း လည်ပတ်မှု အရေအတွက်ပြသခြင်း
- 👥 IP လိပ်စာအလိုက် နောက်ဆုံး ၁၀ ခု လည်ပတ်မှုပြသခြင်း

## 🚀 အသုံးပြုနည်း

1. ဒေါင့်များ `[ ]` နှင့် ကွင်းစကွင်းပိတ်များ `( )` ပါသော လင့်ခ်ကို ထည့်သွင်းပါ
2. "ဒေါင့်များနှင့် ကွင်းစကွင်းပိတ်များ ရှင်းရန် နှိပ်ပါ" ခလုတ်ကို နှိပ်ပါ
3. ရှင်းလင်းပြီး လင့်ခ်ကို ကူးယူပါ

## 🛠️ နည်းပညာများ

- **Frontend:** HTML, JavaScript, Tailwind CSS
- **Backend:** PHP
- **Hosting:** GitHub Pages (Frontend), Custom PHP Server (Backend)
- **Data Storage:** JSON

## 📁 ဖိုင်များ

- `index.html` - ပင်မဝဘ်စာမျက်နှာ
- `track_visit.php` - လည်ပတ်မှု စာရင်းဇယား API
- `visits.json` - လည်ပတ်မှု အချက်အလက်များ သိမ်းဆည်းရာ
